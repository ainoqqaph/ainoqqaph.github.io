全自動化跨國市場情報與 AI 戰情儀表板

2026 / 03 - 迄今

專案目標： 打造從底層資料抓取、AI 語意分析、資料庫清洗，到前端視覺化的「端到端 (End-to-End)」全自動數據決策系統。

使用技術：

Python (Selenium) Docker & Airflow SQL Server Google Gemini API Power BI & DAX
  • 自動化爬蟲： 於 Linux Docker 環境建置爬蟲。導入「崩潰自動重啟機制」，並透過 Apache Airflow 部署每日定時 DAG 排程。
  • AI 賦能數據治理： 串接 Gemini API，將 Google Trends 關鍵字賦予「商業意圖 (Search Intent)」與「產業分類」，將非結構化資料轉化為高含金量的商業指標。
  • POWER BI 視覺化建模： 遵循星狀結構 (Star Schema) 建置資料庫。運用進階 DAX 語法，完美整合四象限散佈圖、神經網絡圖與 AI 戰情快報，打造決策儀表板。
